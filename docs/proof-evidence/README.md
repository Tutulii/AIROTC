# AIR OTC Proof Evidence

This directory contains small, sanitized evidence summaries that are safe to include in the public devnet submission bundle.

It intentionally does not include local logs, `.env` files, private keys, sealed encrypted payload dumps, dependency folders, build outputs, or generated runtime artifacts.

Use these files together with:

- [SUBMISSION_README.md](../../SUBMISSION_README.md)
- [PROJECT_STATUS.md](../../PROJECT_STATUS.md)
- [docs/EVIDENCE_REGISTRY.md](../EVIDENCE_REGISTRY.md)
- [docs/HOW_TO_VERIFY.md](../HOW_TO_VERIFY.md)
